Aide pour les Groupes

À gauche vous pouvez parcourir la liste des groupes existants et en sélectionner un en cliquant dessus.
Au centre apparaîtra la liste des utilisateurs du groupe et leur rôle (Étudiant/ Référent/ Administrateur).
À droite vous pouvez effectuer actions suivantes :

Créer un groupe

Cliquez sur “Créer un groupe”.
Remplissez le nom du groupe, ajoutez une description si nécessaire, sélectionnez l'endroit où vous souhaitez l'ajouter puis validez.

Modifier un groupe

Sélectionnez le groupe que vous souhaitez modifier.
Cliquez sur “Modifier le groupe”, modifiez les champs souhaités puis validez.

Supprimer un groupe

Sélectionnez le groupe que vous souhaitez supprimer.
Cliquez sur “Supprimer le groupe” puis validez.

Purger les groupes

Cliquez sur “Purger les groupes” puis validez.

Ajouter un utilisateur à un groupe

Sélectionnez le groupe dans lequel vous souhaitez ajouter un utilisateur.
Cliquez sur “Ajouter des utilisateurs”.
Dans le champ de recherche, entrez le nom de l'utilisateur à ajouter, sélectionnez cet utilisateur dans le menu déroulant qui apparaît en dessous puis validez.

Pour avoir accès à plus de fonctionnalités, cliquez sur “Plus”. Vous pourrez alors effectuer les actions suivantes :

Limiter les permissions d'un groupe

Sélectionnez le groupe dans lequel vous souhaitez effectuer l’action.
Cliquez sur “Limitations du groupe”.
Si vous voulez modifier vous-même les limitations, modifiez les champs souhaités, puis validez.
Si vous souhaitez appliquer les limitations par défaut, cliquez sur “Valeurs par défaut”.

Partager une machine virtuelle dans un groupe

Sélectionnez le groupe dans lequel vous souhaitez effectuer l’action.
Cliquez sur “VMs pré-configurées”.
Sélectionnez la machine virtuelle que vous voulez partager puis validez.

Supprimer une machine virtuelle partagée dans un groupe

Sélectionnez le groupe dans lequel vous souhaitez effectuer l’action.
Cliquez sur “VMs pré-configurées”.
La liste des machines virtuelles à disposition du groupe s’affiche. Dans la colonne Actions, cliquez sur la croix à côté de la machine que vous voulez supprimer.

Ajouter un réseau dans un groupe

Sélectionnez le groupe dans lequel vous souhaitez effectuer l’action.
Cliquez sur “Réseaux”.
Sélectionnez le réseau que vous voulez rendre disponible puis validez.

Supprimer un réseau d’un groupe

Sélectionnez le groupe dans lequel vous souhaitez effectuer l’action.
Cliquez sur “Réseaux”.
La liste des réseaux disponibles s’affiche. Dans la colonne Actions, cliquez sur la croix à côté du réseau que vous voulez supprimer.

Éteindre les machines virtuelles d’un groupe

Sélectionnez le groupe dans lequel vous souhaitez effectuer l’action.
Cliquez sur “Éteindre les VMs” puis validez.

Si vous sélectionnez un utilisateur dans le panneau du milieu, vous pouvez également effectuer les actions suivantes :

Supprimer un utilisateur d’un groupe

Sélectionnez l’utilisateur que vous souhaitez supprimer du groupe.
Cliquez sur “Supprimer l’utilisateur”.

Modifier les droits d’un utilisateur

Sélectionnez l’utilisateur dont vous souhaitez modifier les droits.
Cliquez sur “Modifier l’utilisateur”, choisissez le statut que vous voulez donner à cet utilisateur dans le menu déroulant puis validez.
