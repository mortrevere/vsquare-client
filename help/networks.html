Aide pour les réseaux

À gauche vous pouvez parcourir la liste des groupes de réseaux existants et en sélectionner un en cliquant dessus.
Au centre apparaîtra la liste des réseaux présents dans ce groupe.
À droite vous pouvez effectuer les actions suivantes :

Créer un réseau

Cliquez sur “Créer un réseau”.
Renseignez le nom du réseau que vous voulez créer, le nombre de ports puis validez.

Modifier les paramètres d’un réseau

Sélectionnez le réseau sur lequel vous souhaitez effectuer l’action.
Cliquez sur “Modifier le réseau”.
Modifiez les paramètres souhaités puis validez.

Supprimer un réseau

Sélectionnez le réseau sur lequel vous souhaitez effectuer l’action.
Cliquez sur “Supprimer le réseau” puis validez.
