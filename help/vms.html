Aide pour les VM

À gauche vous pouvez parcourir la liste des groupes de machines virtuelles existants et en sélectionner un en cliquant dessus.
Au centre apparaîtra la liste des machines virtuelles présentes dans ce groupe.
À droite vous pouvez effectuer les actions suivantes :

Créer une machine virtuelle

Cliquez sur “Créer une VM”.
Renseignez le nom de la machine virtuelle, sélectionnez son fichier OVA ou ISO puis validez.

Créer une machine virtuelle à partir d’une machine préconfigurée

Cliquez sur “Créer une VM préconfigurée”.
Sélectionnez la machine virtuelle à partir de laquelle vous souhaitez créer la vôtre, renseignez le nom de la nouvelle machine virtuelle et une éventuelle description puis validez.

Si vous cliquez sur une machine virtuelle au milieu, vous allez accéder à son détail et vous pourrez alors effectuer les actions suivantes sur cette machine :

Allumer une machine virtuelle

Sélectionnez la machine virtuelle que vous souhaitez allumer.
Cliquez sur “Allumer”.

Supprimer une machine virtuelle

Sélectionnez la machine virtuelle que vous souhaitez supprimer.
Cliquez sur “Supprimer”.

Modifier les paramètres d’une machine virtuelle

Sélectionnez la machine virtuelle dont vous souhaitez modifier les paramètres.
Cliquez sur “Paramètres”.
Si vous voulez modifier vous-même les paramètres, renseignez les champs souhaités puis validez.
Si vous souhaitez appliquer les paramètres par défaut, cliquez sur “Valeurs par défaut”.

Revenir à un snapshot

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action, vérifiez qu’elle est éteinte.
Cliquez sur “Snapshots”.
Sélectionnez le snapshot auquel vous voulez revenir puis cliquez sur “Revenir au snapshot”.

Supprimer un snapshot

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action.
Cliquez sur “Snapshots”.
Sélectionnez le snapshot que vous voulez supprimer puis cliquez sur “Supprimer le snapshot”.

Créer une machine virtuelle à partir d’un snapshot

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action, vérifiez qu’elle est éteinte.
Cliquez sur “Snapshots”.
Sélectionnez le snapshot à partir duquel vous voulez créer la machine virtuelle puis cliquez sur “Créer une VM”.
Entrez le nom de la nouvelle machine virtuelle puis validez.

Créer un snapshot d’une machine virtuelle

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action.
Cliquez sur “Snapshots” puis sur “Créer un snapshot”.
Entrez le nom du snapshot, une éventuelle description puis validez.

Agrandir un disque d'une machine virtuelle

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action, , vérifiez qu’elle est éteinte.
Cliquez sur “Disques”.
Sélectionnez le disque que vous voulez modifier, modifiez sa capacité puis validez.

Supprimer un disque d'une machine virtuelle

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action, vérifiez qu’elle est éteinte.
Cliquez sur “Disques”.
Sélectionnez le disque que vous voulez supprimer puis cliquez sur “Supprimer le disque”.

Créer un dique sur une machine virtuelle

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action, vérifiez qu’elle est éteinte.
Cliquez sur “Disques” puis sur “Créer un disque”.
Renseignez le nom du disque et sa capacité puis validez.

Ajouter un réseau

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action.
Cliquez sur “Réseaux”.
Sélectionnez le réseau que vous voulez ajouter puis cliquez sur “Ajouter une interface”.

Connecter une machine virtuelle à un réseau

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action.
Cliquez sur “Réseaux”.
Un tableau avec les réseaux disponibles s’affiche. Dans la colonne Actions, cliquez sur “Connecter au réseau” en face du nom du réseau auquel vous voulez connecter la machine virtuelle.

Déconnecter une machine virtuelle d'un réseau

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action, vérifiez qu’elle est allumée.
Cliquez sur “Réseaux”.
Un tableau avec les réseaux disponibles s’affiche. Dans la colonne Actions, cliquez sur “Déconnecter” en face du nom du réseau que vous voulez déconnecter.

Supprimer une interface d’une machine virtuelle

Sélectionnez la machine virtuelle sur laquelle vous souhaitez effectuer l’action, vérifiez qu’elle est éteinte.
Cliquez sur “Réseaux”.
Un tableau avec les réseaux disponibles s’affiche. Dans la colonne Actions, cliquez sur “Supprimer l’interface” en face du nom du réseau que vous voulez supprimer de la machine virtuelle.

Pour avoir accès à plus de fonctionnalités, cliquez sur “Plus”. Vous pourrez alors effectuer les actions suivantes :

Cloner une machine virtuelle

Sélectionnez la machine virtuelle que vous souhaitez cloner.
Cliquez sur “Cloner”, renseignez le nom du clone puis validez.

Exporter une machine virtuelle

Sélectionnez la machine virtuelle que vous souhaitez exporter, vérifiez qu’elle est éteinte.
Cliquez sur “Exporter”.

Créer une machine virtuelle préconfigurée

Sélectionnez la machine virtuelle que vous souhaitez mettre à disposition en tant que machine virtuelle préconfigurée, vérifiez qu’elle est éteinte.
Cliquez sur “VM préconfigurée”.
Entrez le nom de la machine, une éventuelle description puis validez.
